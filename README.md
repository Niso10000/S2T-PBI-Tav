# S2T Documentation — דוח תברואה (Power BI)

פיתחתי תיעוד טכני מלא (S2T) לדוח Power BI של מחלקת תברואה עירונית. הדוח מכיל 15 עמודים ומחובר ל-4 מערכות מידע שונות — מעקב GPS, חיישני IoT, ניהול מכולות, ושקילת פסולת — בנוסף ל-CRM ומאגר אירועים. מכיוון שקובץ ה-PBIX אינו ניתן לקריאה ישירה, פירקתי אותו, פיענחתי את ה-JSON הפנימי, וכתבתי סקריפטים ב-Python שחלצו את כל המידע הרלוונטי — טבלאות, שדות, מדידות, פילטרים וויז'ואלים. התוצר הסופי הוא מסמך HTML אינטראקטיבי המתאר את כל שכבות הדאטה, מודל הנתונים, לוגיקת DAX, ה-RLS, ומפת מקורות הנתונים — הכל ב-repo ב-GitHub.
